# `docs/images/`

## Propósito

Esta pasta reúne **evidências visuais** do laboratório: principalmente **provisionamento** no **Azure Portal** (**API Management**, **Application Insights** ou recursos associados). Não é necessário incluir capturas de execução de API quando o escopo do marco for só infraestrutura e documentação.

## Imagens atuais

| Arquivo | Descrição breve |
|---------|------------------|
| **`apim-creating.png`** | Contexto de criação ou configuração do Azure API Management no portal (recorte do fluxo do lab). |
| **`resources-apim-appinsights.png`** | Visão de recursos relacionados APIM e **Application Insights** (ou monitoramento associado) no resource group. |

Novas capturas devem seguir um nome descritivo em minúsculas com hífens (ex.: `apim-products-2025.png`) para facilitar referência no README principal.

## Dados sensíveis

Antes de `git commit` ou de publicar o repositório, **revise cada imagem**: borre ou recorte trechos que exponham **subscription ID**, **tenant ID**, chaves, tokens, connection strings ou informações pessoais visíveis nas barras de URL ou nas lâminas do portal. O objetivo é manter o portfólio seguro e profissional.

O **README raiz** do lab pode linkar estas imagens quando o relatório estiver completo.
